# React + Vite App

This project is a simple web application built with React and Vite.

## Getting Started

To run the project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-react-vite-app.git
2. Change Directorry:

   ```bash
   cd ./frontend
3. Install

     ```bash
     npm install
4. Start the project

     ```bash
     npm run dev
